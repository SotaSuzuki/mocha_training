mocha training
====

Training to write unit tests with using mocha.js

### Description

Let's polish the technique to write unit tests.

### Requirement

- node.js 6.2.1
- npm 3.9.3

### Usage

- Run test

Using mocha.

```shell
$ npm run test
```

### Install

`git clone git@github.com:sota1235/mocha_training.git && npm i`
