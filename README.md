ES2015 Starter Kit
====

Setting files for writing ES2015.

### Description

You can start writing ES2015 right now.

### Requirement

- node.js 6.2.1
- npm 3.9.3

### Usage

- Compile to ES5

```shell
$ npm run build
```

- Run test

Using mocha.

```shell
$ npm run test
```

- Check style

Using ESLint and config of Aribnb.

```shell
$ npm run lint
```

### Install

`git clone {hoge} && npm i`
